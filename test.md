# 조합

생성일: 2023년 3월 24일 오후 11:03

1. 조합(Combination)
    - 서로 다른 n개 중에서 r개를 선택하는 경우의 수 (순서 X, 중복 X)
        
        ex) 서로 다른 4명 중 주번 2명 뽑는 방법

2. 중복 조합
    - 서로 다른 n개 중에서 r개를 선택하는 경우의 수 (순서 X, 중복 O) → 여기서 뜻하는 중복은 조합의 중목을 뜻함.
        
        예시) 후보 2명, 유권자 3명일 때 무기명 투표 방법
